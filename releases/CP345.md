# CP345 — workspace drop choice polish

CP345 keeps CP344 behavior but improves the Open/Merge dialog styling and copy.
The dialog should feel like a Tiinex workflow card rather than a browser or system prompt, with readable muted text and explicit option cards.

- No change to the Open versus Merge decision model.
- Narrow polish pass on visual hierarchy, spacing, copy, and option-card clarity.
