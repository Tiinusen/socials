# CP331 — mobile reclaim and parent scope

CP331 makes mobile reading chrome reclaim real layout space when it fades out, and adds an explicit continuity-parent scope to the artifact wizard. Storage placement remains separate from graph continuity: users can choose a folder through the tree picker and choose/detach a parent through the existing artifact picker pattern. Hidden or unresolved parents remain relation state; changing a parent does not force hidden discovery findings into the visible feed.

This package is the full Tiinex/site source repo shape. Replace the repository contents except `.git/`, then commit and push. Public build output is intentionally excluded.
